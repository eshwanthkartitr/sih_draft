# Quick Start Guide

### Clone the Repo
```bash
git clone sih_draft
cd sih_draft
```
Start Python Server
Ensure the Python server is running:

```bash
python cors_server.py
```
Run the App

```bash
npm run dev
```

Access Locally
Open your browser at http://localhost:5173.
